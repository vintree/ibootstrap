### ibootstrap针对移动端开发的UI库，针对移动端特点添加了许多新特性；

特点：

1、放弃了媒体查询

2、采用rem作为单位

3、弹性盒模型布局

4、针对移动端样式

4、参考了bootstrap用法，使用简单

5、使用sass语法

6、es6特性

7、可自定义webpack打包成AMD、CMD、import使用方法

8、使用gulp集中打包成最后文件

9、基于ibootstrap开发出项目级组件

### 使用前请提前导入jquery或zepto
